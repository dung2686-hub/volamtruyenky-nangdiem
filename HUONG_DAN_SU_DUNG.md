# 🚀 Antigravity Kit v6.0 — Hướng Dẫn Sử Dụng

Đây là bộ **Toolkit** dành cho AI Coding Agent. Khi tích hợp vào dự án, AI sẽ thông minh hơn, code chuẩn hơn, biết review, biết bảo mật, và tự học từ kinh nghiệm.

---

## 📦 Cài đặt (30 giây)

**Bước 1:** Copy thư mục `.agent` vào **thư mục gốc** của dự án.
**Bước 2:** Xong! AI tự động đọc và sẵn sàng.

> Không cần cài thêm gì. Không cần terminal. Không cần npm.

## 💬 KHÔNG NHỚ LỆNH? NÓI TIẾNG VIỆT CŨNG ĐƯỢC!

Bạn **không bắt buộc** phải nhớ lệnh. Cứ nói bằng tiếng Việt bình thường:

| Bạn nói | AI sẽ tự hiểu |
|---|---|
| "giúp mình nghĩ ý tưởng cho trang web bán hàng" | → Tự chạy brainstorm |
| "lập kế hoạch làm tính năng thanh toán" | → Tự chạy plan |
| "code bị lỗi, fix giùm" | → Tự chạy debug |
| "review code cho mình" | → Tự kích hoạt code-reviewer |
| "kiểm tra bảo mật trước khi deploy" | → Tự chạy security check |
| "refactor code cho gọn" | → Tự kích hoạt refactorer |
| "tìm thư viện nào xử lý QR code tốt" | → Tự kích hoạt researcher |

> **Mẹo:** Lệnh `/slash` chỉ là shortcut. Nói tiếng Việt bình thường AI cũng hiểu.


```
1. /code-review     → Review lần cuối
2. /security-check  → Quét bảo mật
3. /deploy          → Triển khai
```

## ⚡ NHỮNG GÌ AI TỰ ĐỘNG LÀM (bạn không cần nhớ)

Dù bạn không gọi, AI **tự động** thực hiện:

- ✅ **Viết code sạch** — theo chuẩn Immutability, KISS, DRY
- ✅ **Xử lý lỗi đúng cách** — không bao giờ nuốt lỗi im lặng
- ✅ **Validate input** — luôn kiểm tra dữ liệu đầu vào
- ✅ **Commit chuẩn** — dùng conventional commits (feat:, fix:, ...)
- ✅ **Tìm giải pháp có sẵn** — trước khi viết code mới từ đầu
- ✅ **Hỏi bạn trước** — không code bừa khi yêu cầu phức tạp
- ✅ **Dùng Python rules** — khi thấy file .py
- ✅ **Dùng Web rules** — khi thấy file .ts/.js/.tsx

## 💡 MẸO

1. **Luôn bắt đầu với `/brainstorm` hoặc `/plan`** cho tính năng lớn. Đừng bảo AI code ngay.
2. **Dùng `/code-review` trước khi merge** — AI sẽ bắt được bug bạn không thấy.
3. **Dùng `/security-check` trước deploy** — đảm bảo không lộ mật khẩu, API key.
4. **Cuối buổi, gõ `/learn`** — AI rút bài học, lần sau làm tốt hơn.
5. **Không nhớ lệnh? Cứ nói tiếng Việt!** AI đủ thông minh để hiểu.
